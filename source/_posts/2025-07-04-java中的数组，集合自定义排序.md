一般在写算法题时，想要对数组，集合等数据结构自定义排序，该文章对其自定义排序进行了总结

# 二维数组自定义排序

例如现在需要根据二维数组的第一个元素排序

1.使用重写compare方法实现

```java
class WorkTestApplicationTests {
    @Test
    public void testCompare(){
        int[][]arr = new int[][]{{1,4},{6,3},{3,4},{5,6}};
        System.out.println("排序前："+Arrays.deepToString(arr));
        Arrays.sort(arr, new Comparator<int[]>() {
            @Override
            public int compare(int[] o1, int[] o2) {
                //根据二维数组中每个元素的第一个值进行升序排序
                return o1[0]-o2[0];
            }
        });
        System.out.println("排序后"+Arrays.deepToString(arr));
    }
}
```

输出结果：

```java
排序前：[[1, 4], [6, 3], [3, 4], [5, 6]]
排序后：[[1, 4], [3, 4], [5, 6], [6, 3]]
```



2.使用lambda表达式实现（和重写compare同理）

```java
class WorkTestApplicationTests {
    @Test
    public void testCompare(){
        int[][]arr = new int[][]{{1,4},{6,3},{3,4},{5,6}};
        System.out.println("排序前："+Arrays.deepToString(arr));
        Arrays.sort(arr, (o1, o2) -> o1[0]-o2[0]);
        System.out.println("排序后："+Arrays.deepToString(arr));
    }
}
```

输出结果：

```java
排序前：[[1, 4], [6, 3], [3, 4], [5, 6]]
排序后：[[1, 4], [3, 4], [5, 6], [6, 3]]
```



# 集合list自定义排序

1.使用compare方式

对list中的元素升序排序

```java
class WorkTestApplicationTests {
    @Test
    public void testCompare(){
        List<Integer>list = new ArrayList<>();
        list.add(4);
        list.add(1);
        list.add(3);
        list.add(8);
        list.add(5);
        System.out.println("排序前："+list);
        Collections.sort(list, new Comparator<Integer>() {
            @Override
            public int compare(Integer o1, Integer o2) {
                return o1-o2;
            }
        });
        System.out.println("排序后："+list);
    }
}
```

输出结果：

```
排序前：[4, 1, 3, 8, 5]
排序后：[1, 3, 4, 5, 8]
```



2.使用lambda方式

```
class WorkTestApplicationTests {
    @Test
    public void testCompare(){
        List<Integer>list = new ArrayList<>();
        list.add(4);
        list.add(1);
        list.add(3);
        list.add(8);
        list.add(5);
        System.out.println("排序前："+list);
        Collections.sort(list, (o1, o2) -> o1-o2);
        System.out.println("排序后："+list);
    }
}
```

输出结果：

```
排序前：[4, 1, 3, 8, 5]
排序后：[1, 3, 4, 5, 8]
```



## 根据集合中的对象属性排序

1.使用compare方式

定义一个user类，此时需要根据age属性排序

```java
class WorkTestApplicationTests {
    @Data
    @AllArgsConstructor
    class User{
        String name;
        int age;
    }
    @Test
    public void testCompare(){
        List<User>list = new ArrayList<>();
        list.add(new User("张三",45));
        list.add(new User("李四",28));
        list.add(new User("王五",16));
        list.add(new User("赵六",66));
        System.out.println("排序前："+list);
        Collections.sort(list, new Comparator<User>() {
            @Override
            public int compare(User o1, User o2) {
                return o1.age-o2.age;
            }
        });
        System.out.println("排序后："+list);
    }
}
```

输出结果：

```
排序前：[WorkTestApplicationTests.User(name=张三, age=45), WorkTestApplicationTests.User(name=李四, age=28), WorkTestApplicationTests.User(name=王五, age=16), WorkTestApplicationTests.User(name=赵六, age=66)]

排序后：[WorkTestApplicationTests.User(name=王五, age=16), WorkTestApplicationTests.User(name=李四, age=28), WorkTestApplicationTests.User(name=张三, age=45), WorkTestApplicationTests.User(name=赵六, age=66)]

```
